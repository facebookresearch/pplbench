Get Started

  1. Download PPLBench:

    via git:
    git clone https://github.com/facebookresearch/pplbench.git

  2. Install PPLBench Core:

    pip install -r requirements.txt

  3. Install PPLs to benchmark:

    Example #1, Stan:
    pip install pystan
    Example #2, Jags:
    pip install pyjags

  4. Run PPLBench:

    Example for Stan and Jags:
    python PPLBench.py -m robust_regression -l jags,stan -k 5 -n 2000 -s 500 --trials 2

  5. Extra: See PPLBench's supported models, PPL implementantations and commands:

    python PPLBench.py -h
